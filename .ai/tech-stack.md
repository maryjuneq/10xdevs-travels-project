# Technology Stack

## Frontend – Astro with React for interactive components
- Astro 5 enables building fast, high-performance sites and applications with minimal JavaScript.
- React 19 provides interactivity where required.
- TypeScript 5 delivers static typing and better IDE support.
- Tailwind 4 allows convenient styling of the application.
- Shadcn/ui offers an accessible React component library on which we will base the UI.

## Backend – Supabase as a comprehensive backend solution
- Supplies a PostgreSQL database.
- Provides SDKs in multiple languages, serving as Backend-as-a-Service.
- Open-source solution that can be hosted locally or on a private server.
- Includes built-in user authentication.

## AI – Communication with models via the Openrouter.ai service
- Access to a broad range of models (OpenAI, Anthropic, Google, and many others) enabling us to find a high-efficiency, low-cost solution.
- Allows setting financial limits on API keys.

## CI/CD and Hosting
- GitHub Actions for creating CI/CD pipelines.
- DigitalOcean for hosting the application via a Docker image.

## Testing
- Vitest for unit and integration testing.
- React Testing Library for component testing.
- Playwright for end-to-end, visual regression, and accessibility testing.
